## ⚖️ Размер файла

CLI подкоманда <code><a href="/ru/docs/explanation/bencher-run/">bencher run</a></code>
может быть использована для отслеживания размера файла (т.е. размера бинарного файла) ваших результатов при помощи опции `--file-size`.
Опция `--file-size` ожидает путь к файлу, размер которого будет измерен.
Под капотом, <code><a href="/ru/docs/explanation/bencher-run/">bencher run</a></code>
выводит результаты в [формате JSON метрик Bencher (BMF)][json].
Поэтому рекомендуется явно установить адаптер `json`.

<FileSizeCode />

Собирается мера `file-size` (т.е. `байты (B)`).
Доступно только значение размера файла (т.е. `value`).
Ни `lower_value`, ни `upper_value` не собираются.
Мера `file-size` не создается по умолчанию для всех проектов.
Однако, когда вы используете опцию `--file-size`, эта мера будет автоматически создана для вашего проекта.
