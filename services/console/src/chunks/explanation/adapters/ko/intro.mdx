어댑터는 벤치마크 하네스 출력을 표준화된 JSON, 벤처 메트릭 포맷(BMF)으로 변환합니다.
이 어댑터는 새로운 보고서가 수신될 때 API 서버에서 실행됩니다.
[벤치마킹 개요][벤치마킹 개요]에서 보다 심층적인 설명을 볼 수 있습니다.
그것들은 `--adapter` 옵션으로 [the `bencher run` CLI 하위명령][bencher run]에 대해 지정될 수 있습니다.
어댑터가 지정되지 않은 경우, 기본적으로 [the `magic` adapter][magic adapter]가 사용됩니다.

사용 사례에 가장 구체적인 어댑터를 사용하는 것이 최선입니다.
이것은 가장 정확하고 성능이 좋은 파싱을 제공합니다.
예를 들어, Rust [libtest bench][rust libtest] 출력을 파싱하는 경우,
`magic` 어댑터가 아닌 `rust_bench` 어댑터를 사용해야 합니다.
좋은 비교를 위해 저희의
[Bencher perf 페이지][perf 페이지]를 참조하세요.
