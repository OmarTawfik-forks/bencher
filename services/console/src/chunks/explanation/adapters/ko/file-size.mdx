## ⚖️ 파일 크기

<code><a href="/ko/docs/explanation/bencher-run/">bencher run</a></code> CLI 서브커맨드는
`--file-size` 옵션을 사용하여 배포물의 파일 크기(즉, 이진 크기)를 추적하는 데 사용될 수 있습니다.
`--file-size` 옵션은 크기가 측정될 파일의 파일 경로를 기대합니다.
내부적으로, <code><a href="/ko/docs/explanation/bencher-run/">bencher run</a></code>
은 결과를 [Bencher Metric Format (BMF) JSON][json]으로 출력합니다.
그러므로 `json` 어댑터를 명시적으로 설정하는 것이 좋습니다.

<FileSizeCode />

`file-size` 측정(즉 `bytes (B)`)이 수집됩니다.
파일 크기 값(즉 `value`)만 사용할 수 있습니다.
`lower_value`나 `upper_value`는 수집되지 않습니다.
`file-size` 측정은 모든 프로젝트에 대해 기본적으로 생성되지 않습니다.
그러나, `--file-size` 옵션을 사용할 때 이 측정은 프로젝트에 자동으로 생성됩니다.

[json]: #-json
