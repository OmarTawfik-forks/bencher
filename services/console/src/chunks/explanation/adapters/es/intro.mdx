Los adaptadores convierten la salida del sistema de referencia de pruebas a un JSON estandarizado, Formato de Métrica de Bencher (BMF, por sus siglas en inglés).
Los adaptadores se ejecutan en el servidor API cuando se recibe un nuevo informe.
Vea la [visión general del benchmarking][benchmarking overview] para obtener una explicación más detallada.
Se pueden especificar para el subcomando CLI de [`bencher run`][bencher run] con la opción `--adapter`.
Si no se especifica un adaptador, se utiliza por defecto [el adaptador `magic`][magic adapter].

Es mejor usar el adaptador más específico para su caso de uso.
Esto proporcionará el análisis más preciso y de mejor rendimiento.
Por ejemplo, si está analizando la salida del bench de [libtest de Rust][rust libtest], debería usar el adaptador `rust_bench`, y no el adaptador `magic`.
Vea nuestra
[página de rendimiento de Bencher][perf page] para una buena comparación.
