## 🦀 Rust Iai-Callgrind

Rust Iai Adapter（`rust_iai_callgrind`）需要 [Iai-Callgrind](https://github.com/iai-callgrind/iai-callgrind) 的输出。
将收集`instructions`，`l1_access`，`l2_access`，`ram_access`，`total_accesses` 和 `estimated_cycles`这些度量值。
只有这些度量值（即`value`）是可用的。没有 `lower_value` 和 `upper_value`度量值。
对于所有项目，默认情况下不会创建此适配器的度量值。
但是，当您使用此适配器时，这些度量值将自动为您的项目创建。
