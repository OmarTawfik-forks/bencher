Built-in [benchmark harness adapters][adapters] convert benchmark output into Bencher Metric Format (BMF) JSON.
Custom benchmarking harnesses should output their results as BMF JSON
and use [the `json` adapter][json adapter].
See [how to track custom benchmarks][custom benchmarks] for a full overview.
